### Admin 上传压缩zip文件(批量上传文件)，上传后自动解压文件  2020.02.21                           

### 主要功能

```
1、国际化。支持多国语言。
2、admin单个文件上传、多个文件(直接将文件压缩成zip，不要含目录缩成zip)上传。


```

### 提交
```
1、admin 删除数据库记录，同时删除文件

```
